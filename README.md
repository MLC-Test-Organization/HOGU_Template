# This is the project for Math # Hands On, Grades Up

If you are not reading this in a codespace, create a codespace on main by clicking the green 'Code' button. It may take a while for the codespace to load initially. 


*! Before you begin editing in the codespace, check that your codespace is using the most recent version of Pretext. Check `requirements.txt` AND the terminal by running "pretext --version". If one or both are out-of-date, run "pretext upgrade" first, followed by "pretext update" in the terminal. It may ask you to do a rebuild. 


The source folder is where your worksheets will live. To start, the folder contains three files: 
`source/Sample.ptx` is a sample worksheet with comments, `source/Session1.ptx` and `source/Session2.ptx` are templates to help you get started. 


Refer to the [Converting to Pretext Process document](https://tamucs-my.sharepoint.com/:w:/g/personal/christynakad_tamu_edu/IQA9MLYGQIn2R7bPcGM69SwzAfwe-jw3grSjpJPi1BZmHz0?e=MbaoLU) for understanding the repository and codespace, converting your documents to the Pretext language, and other Pretext inquiries. 


It's a good idea to have the [Pretext Guide PDF](https://pretextbook.org/doc/guide/pretext-guide.pdf) open as you author. This resource is referenced in comments throughout `source/Sample.ptx`.


See the [PreTeXt documentation](https://pretextbook.org/documentation.html) for links to a variety of other resources and the [PreTeXt Examples](https://pretextbook.org/examples.html) for links to a variety of different Pretext projects. The annotated examples are especially helpful.
